# Fullstack React and Node js

Divide in two folders:

> client
> server

* Initialize the server:
```bash
 > npm init
```

* Install express
```bash
 > npm install express
```

* Install cors
```bash
 > npm install cors
```

* Install cors
```bash
 > npm install mysql2
```

* Install nodemon
```bash
 > npm install nodemon
```


### To run the server:
<p>To start server, I need to run nodemon index.js, but I set in package.json a script
to run npm start instead...</p>

```bash
 > npm start
```
